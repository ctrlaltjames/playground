# JAMESlittle Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript.

## Pages

- **Home** — [`index.html`](index.html) — Main portfolio page
- **To Do** — [`todo.html`](todo.html) — Interactive to-do list app
- **Congrats** — [`congrats.html`](congrats.html) — Gradient text animation page

## Deployment

This project is automatically deployed to GitHub Pages on every push to the `main` branch via GitHub Actions.

## Local Development

Simply open [`index.html`](index.html) in a browser, or use a local server:

```bash
npx serve .
```
